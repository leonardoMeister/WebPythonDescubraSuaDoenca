Projeto Remedium 2021.

Do Latim “Remedium” que significa a Cura, a intenção do projeto é lhe ajudar a encontrar a cura, através da ligação das informações necessárias com tecnologia e segurança, todos conhecimentos adquiridos na melhor universidade catarinense UNIPLAC .

  • Acompanhe GitHub

Destaques Projeto.

  • Design Simplificado.

    Arquitetura de navegação planejada para sua facilidade, com componentes e tela adaptadas e multiplataforma.

  • Experiencia de Usuario.

    Uma experiência assíncrona no cadastro de novas informações, com formulários simplificados e inteligentes.

  • Segurança Informação.

    Criptografia de ponta a ponta para os seus dados sensíveis, validação de sessões e tratamento de dados.

  • ISO 27001
